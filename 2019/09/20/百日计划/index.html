海龟制图turtle

有一只机器海龟起始位置在xoy平面上的（0.0）点
先执行代码

import turtle
turtle.forward(15)
turtle.right(25)

他会向前15个像素，接着向右转25度，会画出一条线段来。这个东西是没有后续操作的，都要在没有反馈的情况下直接写入。

turtlescreen就是运动场。

第三天 这里讲的是分支结构

今天的内容就是讲述if,elif,else.之间的联系
这之中最大的区别就是缩进的问题。相同的优先级的缩进一定要相同

from random import randint

face = randint(1, 6)

这里就是随机数的用法，从random中取出randint这个功能

from random import randint

face = randint(1, 6)
if face == 1:
result = ‘唱首歌’
elif face == 2:
result = ‘跳个舞’
elif face == 3:
result = ‘学狗叫’
elif face == 4:
result = ‘做俯卧撑’
elif face == 5:
result = ‘念绕口令’
else:
result = ‘讲冷笑话’
print(result)
